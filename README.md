# FileResQ: File System Recovery & Optimization Tool

## Description
A Python-based utility designed to help users manage file systems by identifying duplicate files to save space and recovering lost text/log files.

## Features
1. **Optimization:** Scans directories to identify duplicate files based on MD5 content hashing.
2. **Recovery:** Simulates data recovery by extracting specific file types (.txt, .log) from a source drive/folder to a secure destination.

## How to Run
1. Ensure Python 3.x is installed.
2. Run the main interface:
   ```bash
   python main.py
